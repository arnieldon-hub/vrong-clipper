Wrong Clipper is a real-time audio distortion plugin built in C++ using JUCE, focused on exploring nonlinear waveshaping beyond conventional soft and hard clipping. Instead of standard transfer curves, it implements asymmetric and experimentally derived functions to produce complex harmonic content and unique saturation behavior.

Designed for both sound design and production use, the plugin emphasizes low-latency performance, precise parameter control, and expressive tonal shaping.
