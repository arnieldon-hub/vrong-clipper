
🎛 VRONG CLIPPER

FL-style soft clipper VST3/AU with real-time waveform shaving visualization

VRONG CLIPPER is a zero-latency soft clipper designed for modern production workflows. It gives you precise control over transient shaping while letting you see exactly what you’re cutting in real time.

⸻

🔥 Features
	•	Soft clipping engine
	•	Inspired by FL Studio soft clipper behavior
	•	Transparent at low levels, aggressive when pushed
	•	Live waveform shave view
	•	Input vs output visualization
	•	Highlights exactly what peaks are being clipped
	•	Real-time feedback while adjusting parameters
	•	Zero latency
	•	No oversampling
	•	Fully real-time performance (ideal for tracking & mastering chains)
	•	Essential controls
	•	Pre Gain – drive signal into the clipper
	•	Threshold – where clipping begins
	•	Post Gain – output level control
	•	Ceiling – hard cap on output
	•	Knee – smoothness of clipping transition
	•	Color – harmonic shaping / tone
	•	Metering
	•	Input & output level meters
	•	Clip indicator with peak shave readout (dB)

⸻

🎯 Use Cases
	•	Punchy drums (kick / snare clipping)
	•	808 shaping without losing weight
	•	Master bus loudness control
	•	Transient control without full limiting
	•	Creative distortion / saturation

⸻

🧠 Philosophy

VRONG CLIPPER focuses on clarity and speed:
	•	No unnecessary features
	•	No hidden processing
	•	No oversampling latency

What you hear is what you see.


📦 Plugin Format
	•	VST3
	•	AU (macOS)

⸻

⚠️ Notes
	•	Designed for zero latency, so no oversampling is used
	•	Heavy clipping will introduce distortion (intended behavior)
	•	Best results come from balancing Pre / Threshold / Ceiling
